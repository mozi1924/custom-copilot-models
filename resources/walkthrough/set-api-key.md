Responses Copilot uses your own OpenAI (or compatible upstream) API key to make remote models available in the picker.

Paste it once, then update or remove it later from the Command Palette.

- `Cmd/Ctrl + Shift + P`: Open the Command Palette
- `Responses: Set API Key`: Set or update your API key
- `Responses: Clear API Key`: Remove your API key
- `Responses: Get API Key`: Open API key page
